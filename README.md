# S.A.M.E LOTO 5/90

Application web progressive (PWA) — Générateur de grilles Loto 5/90.

## 📱 Installer sur téléphone

### Étape 1 — Créer le dépôt GitHub
1. Connecte-toi sur [github.com](https://github.com)
2. Clique sur **"New repository"** (bouton vert)
3. Nom du dépôt : `same-loto` (ou ce que tu veux)
4. Coche **"Public"**
5. Clique **"Create repository"**

### Étape 2 — Uploader les fichiers
Dans ton nouveau dépôt, clique **"uploading an existing file"** et glisse-dépose ces 5 fichiers :
- `index.html`
- `manifest.json`
- `sw.js`
- `icon-192.png`
- `icon-512.png`

Puis clique **"Commit changes"**.

### Étape 3 — Activer GitHub Pages
1. Va dans **Settings** (onglet en haut du dépôt)
2. Dans le menu gauche, clique **"Pages"**
3. Dans **"Branch"**, sélectionne `main` et le dossier `/ (root)`
4. Clique **"Save"**
5. Attends 1–2 minutes, puis ton lien sera affiché :
   `https://TON-NOM.github.io/same-loto/`

### Étape 4 — Installer sur Android
1. Ouvre le lien dans **Chrome**
2. Chrome affiche une bannière **"Ajouter à l'écran d'accueil"** → clique
3. Ou : menu ⋮ → **"Ajouter à l'écran d'accueil"**

### Étape 5 — Installer sur iPhone
1. Ouvre le lien dans **Safari**
2. Clique l'icône **Partager** (carré avec flèche)
3. Choisir **"Sur l'écran d'accueil"**
4. Clique **"Ajouter"**

## ✅ Fonctionnalités
- Fonctionne **hors connexion** (Service Worker)
- Données sauvegardées localement
- Générateur de grilles (10 à 50 numéros)
- Saisie tirage officiel avec auto-focus
- Compteur cumulatif des numéros
- Historique et statistiques
